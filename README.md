## Título de proyecto
Diseño e Implementación de una Aplicación Web para Facilitar el Transporte Compartido entre Estudiantes Universitarios de Lima Metropolitana

## Objetivo
Diseñar e implementar una aplicación web que facilite la coordinación de viajes compartidos entre estudiantes con rutas y horarios similares, promoviendo la colaboración y un traslado más accesible y eficiente.

## Segmentos objetivos
- Estudiantes universitarios que cuentan con movilidad y están dispuestos a compartir sus traslados hacia el campus.
- Estudiantes sin movilidad que enfrentan dificultades de acceso debido a rutas largas, costosas o inseguras.

## Visión del Proyecto
Este proyecto surge como respuesta a las dificultades de transporte que enfrentan muchos
estudiantes universitarios en Lima Metropolitana, especialmente aquellos que viven lejos de
sus centros de estudio y no cuentan con opciones de movilidad económica, rápidas y seguras.

A partir de experiencias personales y de su entorno cercano, el equipo identifica una
oportunidad de mejora mediante el uso colaborativo de vehículos. Así nace la idea de UniRide,
una aplicación web que facilite la organización de viajes compartidos entre estudiantes con
rutas y horarios compatibles.

La startup académica MoviUni tiene como visión promover una movilidad estudiantil más
eficiente, segura y sostenible, fortaleciendo la colaboración entre pares y reduciendo el
impacto ambiental del transporte individual. El alcance del proyecto incluye el diseño,
implementación y validación de una solución digital que conecta conductores y pasajeros
universitarios, fomentando el traslado compartido como alternativa viable y solidaria.
